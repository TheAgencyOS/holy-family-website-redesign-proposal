Our approach 07 / 12
Four moves, ten months

Four moves. One outcome.

  1. 01

    Consolidate 1,601 URLs to approximately 1,200.

    An AI-assisted audit retires stale news, merges 23 duplicate request-for-information forms into one smart form, and resolves 180-plus top-level sections into a clean 8 to 12.

  2. 02

    Migrate Drupal 10 to WordPress on Pantheon.

    A custom block theme, Advanced Custom Fields Pro for content modeling, and a Dev / Test / Live workflow Rick Mitchell's team already knows how to run.

  3. 03

    Integrate Slate via Gravity Forms webhooks.

    Native WordPress forms styled to match the design system, posting directly to Slate's Source Format endpoint in real time. Slate Ping site-wide on day one.

  4. 04

    Build WCAG 2.2 AA from day one.

    Accessibility shifted left into design tokens, the component library, content editor guardrails, and CI-integrated axe-core. Manual screen reader audit on every template.
