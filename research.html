Add exponential backoff. 2. **Error recovery:** Store state to disk so you can resume interrupted runs. 3. **Image dedup:** Use SHA-256 hash to avoid storing the same image twice. 4. **Link rewriting:** The hardest part. Walk every single file and test every link post-migration. 5. **SEO metadata:** If client uses Yoast, that data is in REST API. Preserve it. 6. **Shortcodes:** You will encounter custom shortcodes. Strip them and generate a report. 7. **Build time:** 2000 static pages on Vercel takes 20-30 min.
